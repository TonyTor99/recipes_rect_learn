Запуск бэка
- После слонирования репозитория переходим в папку \backend
- Затем создаем виртуальное окружение и активируем его
- После этого скачиваем библиотеки из requirements.txt командой pip install -r requirements.txt
- Запускаем сервер командой py manage.py runserver
- По ссылке http://127.0.0.1:8000/swagger/ можете посмотреть как общаться с API

Запуск фронта
- Переходим в папку frontend
- Загружаем зависимости командой npm i
- Запускаем фронт командой npx webpack serve --open
- После чего в браузере у вас откроется страница проекта
